Fire Detection and Alarm System

Intelligent Fire Detection and Alert System.

This project presents a real-time fire detection and alarm system that combines Artificial Intelligence (AI) with Internet of Things (IoT) integration. I trained a custom fire detection model using computer vision techniques and the YOLOv8 (You Only Look Once version 8) architecture, deployed through Ultralytics.

The AI model detects fire in video streams with high accuracy and low latency. Once fire is detected, the system triggers an alarm using a physical IoT setup connected through Python, allowing immediate response to potential fire hazards.

This project showcases not only the integration of advanced deep learning models but also the seamless communication between cloud-based AI and edge IoT devices. It demonstrates my capability to bridge the gap between smart detection and real-world physical alerts.

Car Smart - Fight Fire

Car Smart - Fight Fire

An autonomous robotic car designed to detect fire, navigate toward the source, and extinguish it — all without human intervention. Equipped with flame sensors, obstacle avoidance, and rule-based AI, this robot enhances safety and efficiency in fire emergencies.

Fire incidents spread quickly and can endanger lives. Manual firefighting in dangerous areas is risky. This project addresses the need for an intelligent robotic system that can detect fires, approach and extinguish them, and avoid obstacles in real-time.

The goal is to build an autonomous robot that:

  • Detects fire using flame sensors and rule-based AI.
  • Moves toward the fire and activates a water-spraying system.
  • Avoids obstacles while navigating complex environments.
  • Operates independently to reduce risk to human responders.

Key Components:

  • Arduino (Microcontroller)
  • Flame, Ultrasonic & Infrared Sensors
  • DC Motors + Motor Driver
  • Water Pump & Servo Motor
  • Acrylic Frame, Wheels, Jumper Cables
  • Buzzer, Battery, Breadboard

Sorting Waste in Industry

Sorting Waste in Industry

A tech-based solution to tackle Indonesia’s critical waste management problem. This AI-powered system detects and sorts organic and inorganic waste in real-time using computer vision and IoT integration. Built with scalability and accessibility in mind, it offers a smart, safe, and efficient approach to waste handling.

Indonesia ranks as the 5th largest waste-producing country, yet most of its waste sorting is still done manually. This exposes workers to health risks and increases error potential. There's a pressing need for an automated system that is safe, scalable, and accurate in classifying waste types.

The system aims to automate waste classification and tracking using AI & IoT:

  • Real-time object detection with YOLOv8 and OpenCV (85%+ accuracy)
  • Physical sorting via Arduino-based servo motors & IR sensors
  • RESTful API using Express.js to manage data from AI to dashboard
  • Interactive React.js dashboard for live monitoring
  • MongoDB Atlas for storing classification logs and system status

Riy-Chatbot

Riy-Chatbot: Medical AI Q&A Chatbot with LLaMA 3.2

A multilingual AI chatbot for healthcare Q&A, built using fine-tuned LLaMA 3.2 with LoRA and deployed via Flutter. It understands user health queries and provides accurate, context-aware responses using NLP and Hugging Face datasets — a real-world AI solution developed as part of the NLUG course.

Main Features:

This chatbot aims to improve public access to trustworthy medical information. It supports early self-assessment, reduces misinformation, and enhances healthcare communication — without replacing professional advice.

Tomato Detection

Tomato Detection

A computer vision project designed to automate tomato ripeness classification using deep learning. This system detects and classifies tomatoes into unripe, semi-ripe, and fully ripe categories with high accuracy, significantly improving agricultural efficiency and quality control.

Key Features:

  • Utilizes YOLOv8m and YOLOv5m object detection models on the TomatOD dataset.
  • YOLOv8m achieves mAP@50 > 0.85 and strong performance in varied lighting/backgrounds.
  • Confusion matrix shows high accuracy for unripe (0.96) and fully ripe (0.91) detection.
  • Anchor-free YOLOv8m outperforms YOLOv5m in speed, precision, and stability.
  • Real-time detection results displayed with bounding boxes and confidence scores.

Conclusion:
The implementation of YOLOv8m in this project proves its effectiveness in agricultural automation. It reduces human error, supports food supply chains, and contributes to smart farming. Future work includes model fine-tuning, real-time deployment, and system integration into smart farming environments.

This project highlights how AI and deep learning can transform manual agricultural processes into intelligent, automated systems. The results demonstrate the power of computer vision for enhancing accuracy and scalability in food quality assurance.
